# Sales
## Приложение для удобной и эффективной работы с данными о продажах
### Функционалиные возможности работы приложения:
-	Подключение к БД MS Access
-	Просмотр данных
-	Поиск данных 
-	Сортировка данных 
-	Группировка данных
-	Добавление данных
-	Редактирование данных
-	Удаление данных
-	Формирование и экспорт отчётов на сохранение или печать
### Входные данные:
- База данных MS Access, содержащая данные о продажах
- Для корректной работы БД должна обязательно иметь 2 таблицы - "Пользователи" и "Права пользователей"
### Выходные данные:
- Отчет по данным - таблица с выбранными данными в формате txt
- Отчет может быть сохранен или отправлен на печать
### Как запустить через Visual Studio:
- Склонируйте репозиторий
- Откройте файл WinFormsAppSales.slnx
- Запустите программу
### Как склонировать репозиторий:
- Скопируйте ссылку на репозиторий с GitHub. 
- В терминале введите команду git clone <ссылка на репозиторий>
- После выполнения команды в текущей папке появится папка с тем же именем, что и сам репозиторий. Если нужно указать другую папку, в конце команды для клонирования нужно указать её название: git clone <ссылка на репозиторий> <папка>
### Руководство пользователя (Пример использования):
- Запустите приложение
- Загрузите базу данных (для корректной работы БД должна обязательно иметь 2 таблицы - "Пользователи" и "Права пользователей")
- Пройдите процедуру аутентификации если вы уже есть в базе, или зарегестрируйтесь, получив уровень доступа "Чтение"
- В появившемся выпадающем списке выбеерите таблицу с данными
- После этих действий вы можете пользоваться функциями приложения в соответствии с правами доступа
- Пример как получить данные, отсортированные по столбцу "СтоимостьЕдиницы" по возрастанию:
  - Зайдите в раздел "Обработка данных"
  - Вы можете увидеть таблицу с исходными выбранными данными
  - Зайдите в подраздел "Сортировка"
  - Выберите столбец сортировки из выпадающего списка
  - Выберите направление сортировки из выпадающего списка
  - Нажмите кнопку "Применить"
  - Вернувшись в предыдущий раздел в таблицу вы сможете увидеть отсортированные данные
### Руководство администратора (Как назначать права пользователей):
- Администратор может регулировать права доступа другим пользователям
- Для этого нужно зайти в систему под логином, который помечен в БД уровнем доступа "Администратор" (Код уровня доступа - 1)
- Выбрать таблицу "Пользователи"
- В подразделе "Изменение данных" можно менять права доступа различным пользователям
- Также можно удалять или добавлять пользователей
### Особенности программы:
- Полная валидация ввода (проверка на число, пустой ввод, формат ввода)
- Простой в использовании интерфейс
- Визуализация данных
- Интергация с БД MS Access
- Формирование и экспорт отчетов
### Требования:
- .NET 4.8 или новее
- Visual Studio 2022 (или любая среда с поддержкой C#. Версия языка 12.0 и выше)
- Microsoft Access Database Engine (ACE), который предоставляет OLE DB‑провайдер Microsoft.ACE.OLEDB.12.0
### Как установить Microsoft Access Database Engine:
- Скачайте официальный установщик с сайта Microsoft (см. раздел «Ссылки»)
- Запустите скачанный файл (AccessDatabaseEngine.exe или AccessDatabaseEngine_X64.exe)
- Следуйте инструкциям мастера установки
- Перезапустите приложение
